# Panels

Every function you see on the screen of MinimOSD-Extra is a "Panel".

This panels can be turned on, off and placed anywhere on the screen.

MinimOSD-Extra is is using 3 different screens you can switch between.
Screen 1 Screen 2 and off screen.

You can configure a set of panels to Screen 1:

![](http://gabek.ddns.net/MinimOSD-Extra_wiki_pictures/31.png)

Also a different set of panels to Screen 2:

![](http://gabek.ddns.net/MinimOSD-Extra_wiki_pictures/32.png)

Off screen is always empty.
