Welcome to the MinimOSD-Extra wiki!

MinimOSD-Extra is based on the original ArduCAM OSD project.
We have added extra functions to make FPV flights safer by enableing the pilots to have as many critical information available as possible via the live video from the users plane or copter.

Now the basic functions are the following:

- Changeable unit measurement (US, metric)

- Call sign

- Flight mode

- GPS coordinates

- Heading

- Heading rose

- Real heading

- Home direction

- Home distance

- Pitch

- Roll

- Throttle

- Tune panel

- Visiable Sats

- Air speed

- Ground speed

- Home altitude

- Altitude above sea level

- Battery Percent, or mah used

- Current Draw

- Battery voltage

- Time From Startup and flight time in one panel

- Variometer

- Wind horizontal speed and direction, and short average

- OSD on/off

- Switchable second screen

- Waypoint distance, Waypoint number, direction and Crosstrack error in one panel

- Warning messages for Lost GPS fix, stall, overspeed, battery voltage, battery percent, RSSI (Plane)

- Warning messages for Lost GPS fix, High vertical speed, overspeed, battery voltage, battery percent, RSSI (Copter)

- Efficiency, glide distance & thermic notifier. 3 in one panel (plane)

- Hover time (copter)

- OSD Brightness

- HAM Call Sign

- Smoother horizon

- Real heading

- RSSI

- Temperature

- Trip distance

- After flight summary

This functions can be turned on and off, and placed on different screens now, using the Config. tool.

Also RSSI, switching mode and channel and units of measurement, stall speed warning, overspeed warning, battery warning voltage, battery percent warning, RSSI warning, can be set in the new Config Tool.

We built in a new way of setting video standards. You can set the standard to default to if no video is detected. If video is detected, than it will change it automatically. It is in "Video Mode" menu.
