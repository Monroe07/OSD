# Lost GPS fix

When GPS loses its fix, than this warning will warn you by showing a sign.

This is a fix warning, that means it can't be configured. If warnings are checked than it will be functional.

_The GPS Sats panel also has some warning method as it is. There is 3 different icon. one is for no fix, other is for 2D fix and the last for 3D fix.Also if HDOP is lower then 2 than this icon will blink._
